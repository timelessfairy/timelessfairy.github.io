説明:

Big Horn 18853 Router Speed Controller. Turn the dial and adjust the speed of your router bit. The variable speed dial allows for full horsepower and torque at all speeds from 0 to full RPM. Use with routers, sanders, grinders, drills.

NOTE: Don’t use on BRUSHLESS Motors. Will not work on soft/slow start routers.

Features:

  • Reduces speed without reducing torque
  • Electronic feedback maintains speed by increasing voltage to motor as load increases
  • Heavy Duty Speed Control features better circuitry and higher gauge components
  • Made of Plastic (front) / l (rear) Case for commercial use
  • Features a 3-way rocker switch
  • Use on Brush type tool with 1500W of maximum input power

Specifications:

  • Power Source: AC
  • Voltage: 110 - 120 Volts
  • Amperage: 15 Amp
  • 3-Way Rocker Switch
  • 6 Ft. Cord with Standard 3 Prong Plug
