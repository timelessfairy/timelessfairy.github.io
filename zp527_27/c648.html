Mode control

In any mode, if you hold the key mode button for 3 seconds, you can turn off the light directly, no longer need to click and cycle all modes to turn off it.

Click to switch the lighting mode, long press to directly turn off it, which makes your use easier and more convenient.

1. LED Key Mode

(1)First click, the main strong beam.

(2)Second click, the main weak beam.

(3)Third click, the side strong beam.

(4)Fourth click, the side weak beam.

(5)Fifth click, the red beam.

(6)Sixth click, the SOS red strobe.

(7)Seventh click, power off.

2. Motion Sensor Mode

(1)First click, the main strong beam.
