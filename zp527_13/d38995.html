アメリカで100年の歴史を持つ老舗家電メーカーLASKO社のサーキュレーター。プラスチックで少し大雑把なつくりがアメリカらしさを出します。サーキュレーターとして効力が強く、暑い部屋にこもった空気を窓へ向けて一気に放出したり、梅雨時の洗濯物の部屋干し時にも活躍します。風の音は大きく、きれいで繊細な作りではないですが、その大雑把さがアメリカを感じさせます。重量も軽く持ち運びも楽です。スリムなデザインなので使わない時の収納時も困りません。

- Features -
 
Function 01
Made in USA

100年の歴史を持つラスコ社が製造する、アメリカらしいザックリとした作りが味わい深く魅力的。日本では趣きあるデザイン照明やシーリングファン、レトロファンなどで人気の鎌倉HERMOSA社が代理店を務め国内保証も1年ついています。
 
   
Function 02
強力

軽量なのに、風量は抜群。サーキュレーターとして効力が強く、梅雨時は室内干し時等に活躍します、風の音は大きくいかにもアメリカンな強力な風です。

※イメージ画像はホワイト色のものです。
   
Function 03
風量調整

シンプルなツマミで風量を3段階で調整できます。扇風機のように微弱な風は出ません。

※イメージ画像はホワイト色のものです。
   
Function 04
転倒防止

薄型で軽量の為そのままでは転倒しやすいので転倒防止用のストッパーが付属しています。
   
Function 05
サイズ感

薄型の為収納にも困らず、窓枠の桟などに幅があれば置くこともでき強力な換気扇にもなります。
