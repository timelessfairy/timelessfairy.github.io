説明: The CANADUINO Breadboard Power Supply Module is a huge upgrade from those "regular" modules with their 3.3V and 5V linear voltage regulator.

Our new Breadboard Power Supply modules comes with a USB Type-C port and integrated USB-UART serial interface for programming and monitoring:

Input voltage 5V, 2.1A from standard USB charger.
>90% high-efficiency DC-DC switching converters.
Output voltage 5V, 2A (max.), electronically protected
Output voltage 3.3V, 2A (max.), electronically protected
Output voltage 12V, 0.5A (max.), electronically protected
USB Type-C port for power supply and serial communication.
CH340K USB-UART for programming and serial communication.
Selectable RX/TX logic level (3.3/5V) and 50bps to 2Mbps transfer speed
Reset signal output for easy programming of microcontrollers.

The CANADUINO Breadboard Power Supply Module ships with the 2 spacers as shown in the pictures. These "legs" give it a very stable position and can even be fixed to the surface using M3 screws from the bottom.

Our breadboard fits on standard solderless breadboards like the one in our pictures (breadboard not included).

The voltage selection switches are placed in a way that helps avoiding accidental change of voltages.

The 7-pin header provides the serial connection, reset signal, and the 3 voltages. For example, one power lane can be set to 12V and one to 5V, and you still have the 3.3V available for an external sensor, for example.
