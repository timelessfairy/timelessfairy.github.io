THE NORTH FACE ザ ノースフェイス Red Run Vest レッドランベスト ランニングベスト メンズ 中綿 防寒 保温 2カラー NY82394

ランニング時の運動性能を損なわず、動きやすさと効率的な保温性を両立した高機能保温ベストです。 防風性と耐摩耗性に優れる「PERTEXQUANTUM」をメイン素材に使用し、軽量で保温性が高い「PRIMALOFTBIO」をボール状に集積させた「THERMOBALLBIO」を封入。嵩張ることがなく、効果的な保温力を発揮します。さらに、中わたのTHERMOBALLBIOは、生分解性をもつ100％リサイクル材で、環境にも配慮した素材です。脇には、2WAYストレッチ性と通気性を併せ持つバーサアクティブフリースを採用。タイトなシルエットながらツッパリ感が少なく、優れた運動追従性を追求した作りです。多少の雪や雨に対応するはっ水加工済み。同シリーズの「Red Run Long Pant（NY82395）」とセットアップが可能です。
